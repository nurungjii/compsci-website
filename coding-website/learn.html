The vast amount of resources the Internet contains has led to learning coding on the Internet affordable, reliable, and most importantly, enjoyable. Here are a list of a few Internet Resources that I recommend along with a few key points.



  1. UDEMY
    • A Library Of Courses
    • Covers Every Possible Coding Language And More
    • Always on sale and courses are constantly updated
  2. W3 SCHOOLS
    • Focuses on helping you writing and designing websites
    • Tutorials on HTML and Javascript and CSS styling in websites.
    • Includes excersises and quizzes to help you master the subjects
    • All for free.
  3. CODE ACADEMY
    • Step by Step tutorials and interactive UI to teach a wide variety of Programming
    • Covers the topic of Web Development, Programming and Computer Science, Data Science.
    • Has tutorials on HTML/CSS, Javascript, Python, CMD Line, Python, and even SQL.
    • Free version that can be upgraded later for the premium version.
